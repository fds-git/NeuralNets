В проекте реализованы детекция и распознавание лиц с помощью библиотеки
face-recognition 1.3.0: https://pypi.org/project/face-recognition/
Проект сделан на основе https://projectgurukul.org/deep-learning-project-face-recognition-with-python-opencv/
embeddings.py - скрипт для добавления человека в базу с помощью изображения его лица, полученного с вебкамеры
recognition.py - скрипт для распознавания находящихся в базе лиц на изображении, полученном с вебкамеры
